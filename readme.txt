MUST Hotel Booking provides room availability, reservation management, and booking flows for hotel websites.

== Description ==

MUST Hotel Booking helps hotel and accommodation websites:

* Manage room inventory and availability
* Accept online reservations
* Handle booking checkout and confirmations
* Manage reservations from the WordPress admin area

== Installation ==

1. Upload the `must-hotel-booking` folder to `/wp-content/plugins/`.
2. Activate **MUST Hotel Booking** from the WordPress Admin > Plugins page.
3. Configure rooms, availability, and booking options in the plugin admin screens.
